# TFG — Núcleo de dominio (M1)

Este paquete contiene las entidades del dominio y value objects:
- `tfg/core/enums.py`: enumeraciones (Estado, Modo, TipoCanal, EsquemaCifrado, TipoRecurso)
- `tfg/core/errors.py`: excepciones de dominio
- `tfg/core/value_objects.py`: `Mensaje` y `Resultado`
- `tfg/core/models.py`: entidades: Transferencia, Fragmento, Canal, Endpoint, Credencial, PoliticaCifrado, PerfilRitmo, RecursoDatos

Requisitos: Python 3.11+
